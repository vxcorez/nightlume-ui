# 🌑 Nightlume UI

> Visual Identity & Asset Core for the Nightlume Ecosystem.

This repository serves as the centralized storage for all graphical components, icons, and UI design assets used across Nightlume projects.

## 📂 Repository Structure
* **/icons** - Custom SVG icons and system symbols.
* **/branding** - Logos, wordmarks, and typography guidelines.
* **/wallpapers** - Official environment backgrounds.
* **/previews** - UI/UX high-fidelity mockups.

## 🛠 Tech Stack
* **Vector:** SVG / Adobe Illustrator
* **Raster:** PNG (lossless) / WebP
* **Prototyping:** Figma

## ⚡ Usage
To sync assets with the main project, use the direct CDN links or clone this sub-module into your local `/assets` directory.
